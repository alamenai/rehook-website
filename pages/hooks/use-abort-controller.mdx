# useAbortController

<h3 className="mt-4 text-xl font-normal text-gray-400">A custom hook for managing operations with an AbortController instance. This hook provides functionalities for aborting operations, checking if aborted, and managing abort conditions.</h3>


### Add hook
 
Create a file `use-abort-controller.ts` and copy & paste the code from [useAbortController](/hooks/use-abort-controller#hook).

### Usage

 ```jsx {3,7} copy

import { useEffect, useState } from 'react'
import { useAbortController } from './hooks/use-abort-controller'

const App = () => {

    const abortController = new AbortController()

    const [product, setProduct] = useState<any | null>(null)

    const { abort, isAborted, getSignal } = useAbortController(abortController)

    const signal = getSignal()

    const fetchData = async () => {
        try {
            const response = await fetch('https://fakestoreapi.com/products/1', { signal })
            const data = await response.json()
            setProduct(data)
        } catch (error) {
            if (error instanceof Error) {
                if (error.name === 'AbortError') {
                    console.log('Fetch aborted')
                } else {
                    console.error('Error fetching data:', error)
                }
            }
        } finally {
            console.log(signal.aborted)
        }
    }

    useEffect(() => {
        fetchData()
    }, [signal])

    const handleAbortClick = () => {
        abort()
    }

    const abortStatus = isAborted ? 'You canceled your request' : 'We are processing your request ...'

    const requestStatus = product ? product.title : abortStatus

    return (
        <div
            style={{
                display:"flex",
                flexDirection:"column",
                alignItems:"center",
                position: 'absolute',
                top: '20%',
                left: '45%',
                width: 'auto',
            }}
        >
            <p>{requestStatus}</p>
            {product || isAborted ? null : (
                <button onClick={handleAbortClick} style={{ marginTop: 20 }}>
                    Cancel
                </button>
            )}
        </div>
    )
}

export default App


 ```


### Hook

```ts copy
import { useRef, useState } from 'react'

export const useAbortController = (controller: AbortController) => {
    if (!controller) {
        throw new Error('AbortController instance is required. Make sure to provide a valid AbortController.')
    }

    const [isAborted, setIsAborted] = useState(false)

    const controllerRef = useRef(controller)

    const getSignal = () => {
        return controllerRef.current.signal
    }

    const abort = () => {
        controllerRef.current.abort()
        setIsAborted(true)
    }

    const abortAfter = (time: number) => {
        setTimeout(() => {
            abort()
        }, time)
    }

    const abortIf = (fn: (arg?: unknown) => unknown) => {
        if (fn()) {
            abort()
        }
    }

    const abortIfAfter = (fn: (arg?: unknown) => unknown, time: number) => {
        if (fn()) {
            abortAfter(time)
        }
    }

    return {
        getSignal,
        abort,
        abortAfter,
        abortIf,
        abortIfAfter,
        isAborted,
    }
}

```


## API

[`useAbortController(controller: AbortController): Object`](/hooks/use-abort-controller#hook)

Creates an instance of the useAbortController hook.

 <h1 className='text-xl font-medium mt-6'>Parameters</h1>

| Parameter | Type    | Description                    |
|-----------|---------|--------------------------------|
| `controller`  | AbortController  | The AbortController instance to manage operations.|

[`getSignal(): AbortSignal`](/hooks/use-abort-controller#hook)

Retrieves the AbortSignal associated with the provided controller.

[`abort(): void`](/hooks/use-abort-controller#hook)

Aborts the associated operation.

[`abortAfter(time: number): void`](/hooks/use-abort-controller#hook)

Aborts the associated operation after a specified delay.

 <h1 className='text-xl font-medium mt-6'>Parameters</h1>

| Parameter | Type    | Description                    |
|-----------|---------|--------------------------------|
| `time`    | number  | The delay time in milliseconds.|

[`abortIf(fn: (arg?: unknown) => unknown): void`](/hooks/use-abort-controller#hook)

Aborts the operation if the provided condition is met.


 <h1 className='text-xl font-medium mt-6'>Parameters</h1>


| Parameter | Type                                    | Description                                           |
|-----------|-----------------------------------------|-------------------------------------------------------|
| `fn`      | `(arg?: unknown) => unknown`            | A function that returns a boolean indicating the condition to check. |


[`abortIfAfter(fn: (arg?: unknown) => unknown, time: number): void`](/hooks/use-abort-controller#hook)

Aborts the operation if the provided condition is met after a specified delay.


 <h1 className='text-xl font-medium mt-6'>Parameters</h1>


| Parameter | Type      | Description                                      |
|-----------|-----------|--------------------------------------------------|
| `fn`      | `(arg?: unknown) => unknown` | A function that returns a boolean indicating the condition to check. |
| `time`    | number    | The delay time in milliseconds.                  |

[`isAborted: boolean`](/hooks/use-abort-controller#hook)

A boolean state indicating whether the operation has been aborted.
